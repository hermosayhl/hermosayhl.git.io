Vscode 相对于 sublime 来说，略显笨重，反应速度略慢（但是加上 tabNine 插件后补全直接飞起）。
一款相当好的 Python “IDE”，C++ “IDE”……扩展性十分强，界面也可定制。
今年5月发布的 Remote Development 插件，远程连接服务器，可以远程调试！！！感动！

C++

编译运行
安装 C++ 环境，参考安装 C++

扩展 -> Coderunner -> install 即可运行单文件，不仅局限于 C++，c，也可 Python等

调试

launch.json，修改 "miDebuggerPath"为自己的调试器的

{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "C/C++",
            "type": "cppdbg",
            "request": "launch",
            "program": "${fileDirname}/${fileBasenameNoExtension}.exe",
            "args": ["-exec info registers"],
            "stopAtEntry": false,
            "cwd": "${workspaceFolder}",
            "environment": [],
            "externalConsole": true,
            "MIMode": "gdb",
            "miDebuggerPath": "D:\\software\\C++\\MinGW\\bin\\gdb.exe",
            "preLaunchTask": "g++",
            "setupCommands": [
                {
                    "description": "Enable pretty-printing for gdb",
                    "text": "-enable-pretty-printing",
                    "ignoreFailures": true
                }
            ],
        },
    ]
}

tasks.json

{
    "version": "2.0.0",
    "command": "g++",
    "args": [
        "-g",
        "${file}",
        "-o",
        "${fileDirname}/${fileBasenameNoExtension}.exe"
    ],
    "problemMatcher": {
        "owner": "cpp",
        "fileLocation": [
            "relative",
            "${workspaceRoot}"
        ],
        "pattern": {
            "regexp": "^(.*):(\\d+):(\\d+):\\s+(warning|error):\\s+(.*)$",
            "file": 1,
            "line": 2,
            "column": 3,
            "severity": 4,
            "message": 5
        }
    },
    "group": {
        "kind": "build",
        "isDefault": true
    }
}

设好断点，点击调试（文件路径中不要有中文！！！）

Python

扩展 -> Python

这个插件只是提供 Python 调试等功能的，解释运行 Python 脚本需要系统的python环境（不得不提一下，这个调试功能虽比不上 Pycharm 强大，但 vscode 轻量啊）

插件推荐

1. aync-setting

插件与配置等多终端同步

每次重装电脑，打开 vscode 第一步就是安装这个插件，然后…直接回到之前所有的 vscode 配置，哈哈。
具体配置过程参考这里

2. tabNine

人工智能的自动补全，十分强大

3. Code Runner

能编译运行 C++、Java、Python 等多种语言。十分方便，前提是相应的开发环境已经安装好了

4. Remote Development

远端连接利器，terminal 和 filesystem 集成，在 vscode 调代码，在服务器的 1080ti 上跑代码

5. Bookmarks

打标签，快速访问之前未做完的部分或需要注意的部分，对于代码量较多的程序来说很方便。

6. Material Theme

一款很漂亮的主题

7. Github Plus Theme

类 github 主题

8. vscode-icons-mac

类 mac 的文件图标显示

9. Path Autocomplete

路径自动填写，写较长的绝对路径时很方便

10. GitLens

虽然 vscode 默认集成了 Git 部分，但是可视化程度不高，GitLens 可视化程度很高，信息很详尽
