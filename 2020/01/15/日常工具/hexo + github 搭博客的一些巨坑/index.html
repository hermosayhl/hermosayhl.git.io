404 github page not found

巨坑啊！！！

出现这个结果的原因十有八九是 repo 的名字不对。

官方描述，必须是 username.github.io，也就是说例如我的github用户名是 hermosayhl，那么我申请的repo就必须命名为 hermosayhl.github.io ！！！

不仅如此，github 的用户名还必须是小写，不能有大写字母，巨坑啊，逼得我只能修改我的用户名。

如果还是 404，那就重新生成：

hexo clean
hexo g
hexo s
